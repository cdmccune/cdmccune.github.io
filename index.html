FitSW - Personal Training Softare - Lead iOS Developer

The FitSW iOS app has rating of 4.8 with over 1600 reviews and an average of 1700 monthly users. FitSW is fitness software that enables personal trainers build their businesses by designing workout and nutrition plans. Our software is available on iOS, Android, and Web. As the lead iOS developer, I spearhead the mobile side of every new feature we implement from the first design talks all the way to the finished shipped code. There are several major projects I have taken from start to finish:
- Reworking of the entire nutrition planning side of the app
- Designing and building of an in app calendar and schueduling feature
- Updating of the menu and dashboard to be dynamically built through API call to increase customizability for trainers.

On Your Heart - Emotional Support Application - iOS

The OnYourHeart iOS application was built to help users with the emotional ups and downs of life through the use of the bible. It uses Firebase for cloud storage and accesses multiple REST Api endpoints and parses that data into a unique data structure. While the main portion of the app is built in Swift ith UIKit, there is also an available widget built out using SwiftUI. Submitted through Apple's AppStore submission process, this app is currently live on the app store.
